Inline Source¶

Example configuration (YAML):

  - type: inline
    config:
      instances:
        - instance_id: my_service
          service_clusters:
            - P2
          parameters:
            upstream_address:
              - address: service.domain.com
                region: us-east-1
          plan_id: 7d57270a-0348-58d3-829d-447a98fe98d5
